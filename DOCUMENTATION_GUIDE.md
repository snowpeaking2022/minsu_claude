# 文档使用指南

## 概述

为了提供更好的文档阅读体验，我们创建了一个完整的 HTML 格式文档 `COMPLETE_DOCUMENTATION.md`，它包含了项目的所有文档内容，并提供了以下功能：

1. **侧边栏导航** - 固定在左侧，方便快速跳转到各个章节
2. **页面内跳转** - 点击目录或链接可快速定位到相关内容
3. **响应式设计** - 在桌面和移动设备上都能良好显示
4. **平滑滚动** - 提供更好的阅读体验
5. **代码高亮** - 技术代码具有语法高亮功能
6. **搜索功能** - 支持在浏览器中搜索内容

## 如何查看

### 方法 1: 直接在浏览器中打开（推荐）

1. 在项目根目录找到 `COMPLETE_DOCUMENTATION.md` 文件
2. 右键点击该文件，选择「打开方式」→「浏览器」（Chrome, Firefox, Safari 等）
3. 文档将在浏览器中打开，您可以看到：
   - 左侧的侧边栏导航
   - 顶部的目录导航
   - 主内容区域

### 方法 2: 作为本地 HTML 文件查看

```bash
# 在终端中使用浏览器打开
# macOS
open COMPLETE_DOCUMENTATION.md

# Linux (使用默认浏览器)
xdg-open COMPLETE_DOCUMENTATION.md

# Windows
start COMPLETE_DOCUMENTATION.md
```

### 方法 3: 使用本地服务器

```bash
# 使用 Python 快速启动本地服务器
python3 -m http.server 8000

# 然后在浏览器中访问
# http://localhost:8000/COMPLETE_DOCUMENTATION.md
```

## 使用指南

### 侧边栏导航

1. **固定在左侧** - 始终可见，方便快速导航
2. **点击跳转** - 点击任意章节标题即可跳转到该内容
3. **当前位置高亮** - 页面滚动时，对应的导航项会高亮显示
4. **平滑滚动** - 提供更好的视觉体验

### 内容导航

1. **顶部目录** - 页面顶部有完整的目录导航
2. **锚点链接** - 所有章节标题都有锚点链接
3. **搜索功能** - 使用浏览器的搜索功能（Ctrl/Cmd + F）

### 代码块

1. **复制功能** - 点击代码块右上角可复制代码
2. **语法高亮** - 技术代码具有语法高亮
3. **滚动查看** - 长代码块可横向滚动

### 表格

1. **响应式** - 在小屏幕上会自动调整
2. **单元格对齐** - 内容对齐方式一致
3. **边框设计** - 表格有清晰的边框和分隔线

### 注意事项

1. **信息框** - 有不同颜色的信息框表示重要程度
2. **警告提示** - 黄色框表示警告信息
3. **重要信息** - 红色框表示重要信息
4. **提示信息** - 蓝色框表示提示信息
